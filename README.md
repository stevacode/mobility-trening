# Mobility trening

Interaktivni vodič za mobility rutinu (Ido Portal / Floreio Project, organizacija: Antranik Kizirian).

- Blokovi se šire na tap, vežbe se čekiraju, prsten prati napredak.
- Uz svaku vežbu ▶ vodi na video pokreta.
- Radi u pregledaču, bez servera. Otvori `index.html`.

## Uživo (GitHub Pages)

Kad uključiš Pages (Settings → Pages → Deploy from branch → `main` / root),
stranica je dostupna na:

`https://stevacode.github.io/mobility-trening/`

## Struktura

- Priprema: zagrevanje + ručni zglobovi
- Krug: Squat Clinic 1.0 → ramena → Squat Clinic 2.0 → skapularna mobilnost
- Završetak: gležnjevi
